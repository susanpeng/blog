My First Markdown Next.js Blog

Posted on 2022-02-08

I really enjoy this blog because I can conrol almost everything by myself; I can change the theme, add more tabs to the blog, etc. Wonderful.

To creat a next.js project

npx create-next-app mysite name

To start the blog:

npm run dev

To build the static publishing folder on GitHub

npm run build

Then, I'll get the out folder with all static file in it.
